About SchoolAppUI

SchoolAppUI is a teaching-friendly Angular codebase that mirrors real-world structure: lazy-loaded features, a shared module for declarables, a core module for singletons and interceptors, and JSON-backed endpoints for rapid prototyping.

Architecture
  • Core / Shared / Feature slicing
  • Guards, Interceptors, Services
  • Template + Reactive Forms
  • Routing with preloading & lazy loading
Tech Choices
  • Angular 16
  • Bootstrap 5 (utilities + components)
  • json-server mock API
  • HttpClient + Interceptors
Goals
  • Readable for freshers
  • Extendable to real projects
  • Demonstrate best practices
  • Keep UI simple and neat
Roadmap
  1. Role-based permissions on menus and routes
  2. Pagination & sorting for large datasets
  3. File uploads (e.g., student photos)
  4. Unit tests & e2e samples
